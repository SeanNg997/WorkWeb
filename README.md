# WorkWeb

WorkWeb 是一个**极简风格**、**开源**、**完全本地化**的个人工作面板，专门用于管理便笺、项目内容和个人常用信息。适合喜欢极简风格体验，重视数据安全与隐私的广大学生和打工人。

## 1 功能概览

### 1.1 TODO / 便笺

- 记录待办、灵感、短笔记
- 支持卡片内直接编辑
- 支持 Markdown 内容编辑和展示

![Alt TODO功能](.\image\pic1.png)

### 1.2 项目书架

- 用书架形式管理项目
- 每个项目支持多页内容
- 适合记录计划、过程、总结、资料沉淀

![Alt 项目书架功能](.\image\pic2.png)

### 1.3 常用个人信息

- 按标签整理信息
- 支持加密字段显示
- 适合保存账号、地址、邮箱、备注等内容

![Alt 常用个人信息功能](.\image\pic3.png)

## 2 快速开始

### 2.1 安装运行

前往 GitHub Release 页面下载最新安装包并安装（[Releases](https://github.com/SeanNg997/WorkWeb/releases)），安装后直接运行即可。

### 2.2 源码运行

1. 首先安装 [Node.js](https://nodejs.org/)；

2. 启动源码版：

   - macOS：双击 `Start WorkWeb.command`

   - Windows：双击 `Start WorkWeb.vbs`（启动失败可双击 `Start WorkWeb.bat` 查看错误信息）


## 3 数据存储说明

- 桌面应用首次启动时，会提示确认数据目录；

- 默认数据目录为安装目录下的 `data`，如果继续使用默认目录，卸载软件时数据可能被一并删除；
- 建议把数据目录改到安装目录之外，如果将目录设置在云同步文件夹中，可实现多平台同步。
